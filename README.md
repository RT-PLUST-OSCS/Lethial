## 📖 Overview

**Lethial Web** is a sharp and stealthy web application that simulates a job application form.  
Users are presented with a form they must fill out to "apply for a job." Once submitted, all entered data is silently and automatically sent to the operator's server.

It is built with:

- 🐍 **Python**
- 🌐 **Flask**
- ☁️ **Cloudflare Tunnel (Cloudflared)** – for public access

---

## ✨ Features

- 🎯 Polished job application form interface
- 📨 Silent data capture & server delivery
- 🌍 Optional public access via Cloudflare Tunnel
- 💻 Cross-platform: **Linux**, **macOS**, **Windows**
- ⚡ Lightweight & easy to deploy
